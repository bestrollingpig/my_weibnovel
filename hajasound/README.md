# 종합 한문 발음 편집기

한문(중국어 한자)을 입력하면 **한글 독음**, **북경어(표준중국어) 발음**, **광동어 발음**을 한글로 표기해주는 웹 도구입니다.

## 기능
- 한자 입력 → 한글 독음 자동 변환 (27,000+ 한자)
- 북경어 병음 → 한글 변환 (천지인 방식 4색 성조 표시)
- 광동어 粤拼 → 한글 변환 (4색 성조 통일)
- TTS 음성 재생 (한국어/북경어/광동어)
- 각 글자별 상세 음운 정보

## 사용법
1. [GitHub Pages URL] 접속
2. 상단 입력창에 한문 입력
3. 체크박스로 각 패널 표시/숨김 전환
4. 🔊 버튼으로 발음 듣기

## 파일 구성
| 파일 | 설명 |
|------|------|
| `index.html` | 메인 페이지 |
| `hanja-data.js` | 한자 → 한글 독음 (27,499자) |
| `hanja-pinyin.js` | 한자 → 북경어 병음 (20,934자) |
| `pinyin-to-hangul.js` | 북경어 병음 → 한글 음절 매핑 |
| `hanja-canton.js` | 한자 → 광동어 粤拼 (27,499자) |
| `jyutping-to-hangul.js` | 광동어 粤拼 → 한글 음절 매핑 |

## 참고
- 모든 파일이 오프라인에서 동작합니다
- 성조 색상: 북경어 4성(🔴🟠🟢🔵) / 광동어 6성(🔴🟠🟡🟢🔵🟣)
- 북경어와 광동어는 성조 체계가 다름 (4성 vs 6성)
- 천지인 방식 변환 원리는 화면 하단 "변환 원리 보기" 참고
