따옴표 교정기 (Smart Quotes Fixer)

💡 뒤집힘 및 소스코드 오염 완벽 해결 버전
1. 문맥 분석 엔진: 앞뒤 공백과 조사(은, 는, 이, 가 등)를 분석하여 여는 모양(“ ‘)과 닫는 모양(” ’)이 뒤집히는 현상을 방지합니다.
2. HTML 태그 보호: 텍스트 내부의 HTML 태그 속성값(href="", class="")은 그대로 유지하고 실제 텍스트만 정밀하게 교정합니다.
3. 줄바꿈(개행) 지원: 여러 줄에 걸쳐 작성된 긴 따옴표 대사도 끊김 없이 정확하게 변환합니다.

사용 방법

  1. 아래 입력창에 텍스트 또는 HTML 소스코드를 붙여넣습니다.
  2. 교정하기 버튼을 누릅니다.
  3. 교정된 결과와 문맥 분석 로그를 확인하고 결과 복사를 누릅니다.
