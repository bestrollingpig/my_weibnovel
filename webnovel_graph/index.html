🚀 시작하기

  • 노드 추가: 이름 입력 후 '노드 추가' 또는 빈 공간 더블클릭
  • 노드 선택: 노드 클릭하면 오른쪽 패널에 속성 표시

🔗 관계 연결 (3가지 방법)

① 클릭으로 연결
출발 노드 클릭 → 도착 노드 클릭 → 라벨 입력 → 추가
② Ctrl+드래그
Ctrl 누르고 출발→도착 드래그 → 라벨 입력
③ 드래그 모드
'드래그 모드' 켜고 노드간 드래그로 계속 연결

📸 PNG 캡쳐

  • 해상도 조절: 1x ~ 4x

  • 클립보드 복사: 이미지 복사

🔊 TTS 음성

  • 메모 작성 후 '읽기'
  • 캐릭터 매핑: 철수: 안녕
  • 태그 사용: [나래이터] 안녕

⌨️ 단축키

  • Delete: 선택한 요소 삭제
