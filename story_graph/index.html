📖 소설 관계도 분석기

수동 복사 + 비평 + 효과 · 버전 5.0

🔐 로컬 보안 유지

모든 데이터는 당신의 PC에만 저장됩니다.
외부 서버로 전송되는 데이터는 없으며, 외부 AI로 보내는 텍스트는 당신이 직접 복사/붙여넣기하는 방식입니다.

📌 사용 방법

1
📂 폴더 연결상단 "폴더 선택"으로 소설 파일 폴더를 선택하세요.
2
📄 파일 선택좌측 파일 목록에서 분석할 소설 파일을 클릭하세요.
3
📋 프롬프트 복사"프롬프트+텍스트 복사"로 분석 지시문을 복사하세요.
4
📥 결과 붙여넣기AI 출력(JSON)을 붙여넣고 "그래프 생성"을 클릭하세요.

📊 비평 시스템

  • 비평 탭에서 전용 프롬프트를 복사할 수 있습니다.
  • 웹소설 장르 문법과 독자 친절함을 기준으로 5개 항목을 평가합니다.
  • 점수는 100점 만점이며, 등급(독보적/경쟁력/정체/이탈/해체)으로 표시됩니다.
  • 총평, 핵심 분석, 최강/최약 문장을 함께 제공합니다.

🎨 그래프 조작

  • 노드 드래그 — 마우스로 노드를 끌어 위치 조정 (물리 효과 적용)
  • 휠 줌 — 마우스 휠로 확대/축소
  • 노드 호버 — LLM 위키 스타일의 상세 툴팁 표시
  • 노드 클릭 — 우측 위키 패널에 고정 표시
  • 필터 — 상단 필터로 특정 타입만 표시
  • 🌊 떨림 — 노드가 미세하게 떨리는 효과
  • 🌀 스프링 — 드래그 후 탄성 운동 효과
