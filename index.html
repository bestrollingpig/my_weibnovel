📖 도움말 ▼

🚀 시작하기

  • 노드 추가: 왼쪽 패널에서 이름 입력 후 '노드 추가' 또는 그래프 빈 공간 더블클릭
  • 노드 선택: 노드 클릭하면 오른쪽 패널에 속성 표시
  • 노드 이동: '드래그 모드' 켜고 노드 드래그

🔗 관계 연결 (3가지 방법)

① 클릭으로 연결 (가장 정확함)
1. 출발 노드 클릭 → 자동 입력
2. 도착 노드 클릭 → 자동 입력
3. 관계 라벨 입력 (예: "친구", "적")
4. '연결 추가' 버튼 클릭
② Ctrl+드래그 (가장 빠름)
1. Ctrl(맥 Cmd) 키 누른 상태
2. 출발→도착 노드로 드래그
3. 라벨 입력 후 확인
③ 드래그 모드 (연속 작업)
1. '🖱️ 드래그 모드' 버튼 클릭 (빨간색 깜빡임)
2. 노드간 드래그로 계속 연결
3. 다시 버튼 클릭하여 모드 해제

📸 PNG 캡쳐

  • 해상도 조절: 1x ~ 4x (슬라이더로 조정)
  • 전체 저장: 전체 그래프 PNG 저장
  • 클립보드 복사: 이미지 복사 (Ctrl+V로 붙여넣기)

🔊 TTS 음성 낭독

  • 메모 작성 후 '읽기' 버튼 클릭
  • 캐릭터 매핑: 철수: 안녕하세요 형식
  • 태그 사용: [나래이터] 안녕 [아이 여] 반가워

💾 데이터 관리

  • JSON 업로드/다운로드로 백업
  • URL 공유로 현재 상태 저장

⌨️ 단축키

  • Delete / Backspace: 선택한 요소 삭제
  • Ctrl+드래그: 빠른 연결
  • 더블클릭(빈공간): 노드 추가
