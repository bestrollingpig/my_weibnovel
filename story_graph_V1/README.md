# ⚓ 소설 관계도 분석기

웹소설 분석을 위한 올인원 도구입니다. 로컬에서 안전하게 작동하며, 외부 AI와 연동하여 관계도 분석 및 비평을 수행할 수 있습니다.

## ✨ 주요 기능

### 📝 편집기
- 로컬 폴더의 `.txt`, `.md` 파일 읽기/쓰기
- 실시간 자동 저장 및 백업
- 분할 모드 (참고 파일과 동시 편집)
- 타자기 모드
- 문서 통계 (글자수, 단어수, 읽기 시간 등)

### 🌐 지식 그래프
- Cytoscape.js 기반 인터랙티브 그래프
- 노드 타입별 색상 구분 (인물/사건/장소/아이템)
- 실시간 필터링
- 떨림(Breathe) & 스프링(Spring) 효과
- 다양한 레이아웃 (기본/트리/마인드맵)
- 노드 위키 (설명/이미지/URL 첨부)
- PNG 내보내기 및 클립보드 복사

### 📊 분석 시스템
- **수동 복사 방식**: 외부 AI로 텍스트를 안전하게 전송
- **관계도 분석**: JSON 형식으로 결과를 붙여넣어 그래프 생성
- **웹소설 비평**: 5개 항목 100점 만점 평가
- **레이더 차트**: 시각적 점수 분포 확인
- **자동 JSON 수정**: 잘못된 형식 자동 복구

### 🎵 타자 음악
- 키보드 입력 시 실시간 음악 재생
- 5가지 모드: 키보드/연주곡/즉흥연주/혼합/오디오
- Soundfont 고품질 악기 지원
- MIDI 파일 업로드 및 재생
- 오디오 클리퍼 (업로드한 오디오를 타자음으로 재생)

### 🔧 유틸리티
- 폴더 내 모든 파일 찾기/바꾸기
- 문서 합치기/분할
- 스냅샷 (문서 상태 저장/복원)
- 모바일 프리뷰
- TTS (텍스트 읽어주기)
