🔐 로컬 보안 유지

모든 데이터는 당신의 PC에만 저장됩니다.
외부 서버로 전송되는 데이터는 없으며, 외부 AI로 보내는 텍스트는 당신이 직접 복사/붙여넣기하는 방식입니다.

📌 사용 방법

1
📂 폴더 연결상단 "폴더 선택"으로 소설 파일 폴더를 선택하세요.
2
📄 파일 선택좌측 파일 목록에서 분석할 소설 파일을 클릭하세요.
3
📋 프롬프트 복사"프롬프트+텍스트 복사"로 분석 지시문을 복사하세요.
4
📥 결과 붙여넣기AI 출력(JSON)을 붙여넣고 "그래프 생성"을 클릭하세요.

📊 비평 시스템

  • 비평 탭에서 전용 프롬프트를 복사할 수 있습니다.
  • 웹소설 장르 문법과 독자 친절함을 기준으로 5개 항목을 평가합니다.
  • 점수는 100점 만점이며, 등급(독보적/경쟁력/정체/이탈/해체)으로 표시됩니다.
  • 총평, 핵심 분석, 최강/최약 문장을 함께 제공합니다.

🎨 그래프 조작

  • 노드 드래그 — 마우스로 노드를 끌어 위치 조정 (물리 효과 적용)
  • 휠 줌 — 마우스 휠로 확대/축소
  • 노드 호버 — LLM 위키 스타일의 상세 툴팁 표시
  • 노드 클릭 — 우측 위키 패널에 고정 표시
  • 필터 — 상단 필터로 인물/사건/장소/아이템 타입별 표시
  • 레이아웃 — 기본(cose)/트리(dagre)/마인드맵 전환
  • ⟲ 초기화 — 그래프 위치와 확대율 초기화
  • 📋 내보내기 — 그래프 데이터(JSON)를 클립보드로 복사
  • 🌊 떨림 — 노드가 미세하게 떨리는 효과
  • 🌀 스프링 — 드래그 후 탄성 운동 효과
  • ⛶ 확대 — 그래프를 전체화면으로 보기
  • 📷 저장 / 📋 복사 — 그래프를 PNG 이미지로 저장하거나 클립보드에 복사
  • 해상도 — PNG 내보내기 해상도 배율 설정 (1x~5x)

🖼️ 노드 이미지 첨부

  • 노드 클릭 → 위키 패널의 첨부 URL 입력란에 이미지/유튜브 URL 입력
  • 입력 즉시 그래프 노드 우측 상단에 📎 버튼이 나타납니다
  • 📎 버튼을 클릭하면 이미지 미리보기 또는 유튜브 영상을 캔버스에 표시
  • URL은 .graph.json 파일에 저장되어 새로고침 후에도 유지됩니다

⌨️ 타자 음악

  • 본문 편집기 툴바의 🎵 버튼으로 켜기/끄기
  • 키음 체크박스로 키보드 소리 on/off, 키보드 종류(청축/갈축/적축 등) 선택 가능
  • 사용자 지정 옵션으로 직접 업로드한 소리(물소리, 새소리 등)를 키음으로 사용
  • 배경음악 MP3/WAV 파일 업로드, 타이핑 속도에 따라 음악 재생 속도 자동 연동
  • ⌨ 연동 체크 시 타이핑 속도(RPM)에 맞춰 음악 속도가 실시간 변경
  • 일정 시간 idle 시 음악 볼륨이 자동으로 줄어듦 (덕킹)
